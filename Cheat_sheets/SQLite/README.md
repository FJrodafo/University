## Install

### APT repository

Update the repositories to ensure you have the latest versions of the packages.

```shell
sudo apt update
```

Install SQLite with the following command.

```shell
sudo apt install sqlite3
```

Check the installation to make sure everything went well.

```shell
sqlite3 --version
```
