## Releases

### Zip

```shell
zip -r your-package-v1.0.0.zip LICENSE README.md your-package.sh
```

```shell
unzip -l your-package-v1.0.0.zip
```

### Tar

```shell
tar -czvf your-package-v1.0.0.tar.gz LICENSE README.md your-package.sh
```

```shell
tar -tzf your-package-v1.0.0.tar.gz
```
